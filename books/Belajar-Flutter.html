Belajar Flutter

Muhammad Fajri
Sunday, 11 December 2022

Bahasa pemrograman dengan satu code base yang dapat berjalan secara cross platform (Web, Android, iOS, Windows, Linux, MacOS).

Selain sebagai bahasa pemrograman resmi untuk pengembangan aplikasi Android, Kotlin juga merupakan bahasa yang multiplatform, dapat berjalan di desktop, enterprise, web dan IoT.

Interoperable dengan Java, membuatnya memberikan kemudahan bagi Java developer untuk beralih ke Kotlin dengan penyesuaian yang lebih mudah. Tidak hanya itu, di web dapat diintegrasikan dengan JavaScript dengan meng-import library-library yang diperlukan.
