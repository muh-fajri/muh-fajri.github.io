Perintah Dasar Kotlin

Muhammad Fajri
Sunday, 04 April 2021

Bismillahirrahmanirrahim.

Struktur Dasar

fun main() {
    println("Hello World")
}

Standar Output

// print: menampilkan output tanpa penambahan baris baru
print("Hello ")
print("world!")

// println: menampilkan output dengan penambahan baris baru
println("Hello world!")
println(42)

Struktur Program

package hello

fun main(args: Array<String>) {
    println("Hello world!")
}

Keterangan program:

  • package hello artinya kita membuat sebuah paket bernama hello. Paket dalam kotlin bersifat opsional(boleh ditulis boleh tidak).
  • fun main(){...} adalah fungsi utama program.
  • args: Array<String> adalah variabel yang menyimpan argumen.
  • println() statement atau fungsi untuk menampilkan teks ke layar
  • semicolon (;) pada kotlin bersifat opsional.

Variabel dan Tipe Data

Variabel merupakan simbol yang digunaan untuk menyimpan sebuah nilai. Tipe data adalah jenis nilai yang akan disimpan.

Kotlin mendukung 2 jenis variabel yaitu:

  • Mutable, variabel yang dapat diubah (diberikan ulang nilai). Kata kunci yang digunakan adalah var.
  • Immutable, variabel yang tidak dapat diubah (nilainya tidak dapat diganti). Kata kunci yang digunakan adalah val.

Deklarasi variabel:

var/val namaVariabel : TipeData = Nilai

Contoh

// 1. Mmebuat variabel kosong (tipe data harus dituliskan)
var namaLengkap: String

// 2. Membuat variabel dan langsung diberikan nilai
//    (tipe data tidak wajib dituliskan karena sudah memiliki nilai)
var alamat: String = "Makassar"
var tanggalLahir = "01-01-2021" as string

// 3. Membuat variabel tanpa menuliskan tipe data
var nabaBarang = "Smartphone"
var harga = 2500000
var berat = 0.8

Tipe Data

  • Int (Integer): bilangan bulat, contoh: 12,1_000
  • String (teks), contoh: “belajar kotlin asik”
  • Float (bilangan pecahan), contoh 21.23, 1.3
  • Double (bilangan pecahan), tapi punya ukuran penyimpanan yang lebih besar dari Float.
  • Boolean, tipe data yang hanya bernilai true dan false
  • Char (karakter), contoh: ‘A’
  • Unit, tipe data yang hanya punya satu nilai, yaitu: objekUnit. Tipe data ini mirip seperti void pada Java.

Tipe Konversi Data

Konversi tipe data dilakuka untuk mendapatkan nilai yang akurat.

var a = 14
var b = 3
varc = a.toFloat() / b.toFloat()
print(c) // output 4.66666

Konversi tipe data yang lain:

  • toInt() untuk mengubah ke integer.
  • toString() untuk mengubah ke String.
  • toDouble() untuk mengubah ke Double.
  • toLong() untuk mengubah ke long integer.
  • toShort() untuk mengubah ke short integer.
  • toChar() untuk mengubah ke Char (karakter).
